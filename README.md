# ShowHunt

A TV show discovery app with search, genre filtering, and a personal watchlist. Built with the TVMaze API — free, no key needed.

## Features

- Browse trending shows sorted by rating
- Filter by genre (Drama, Comedy, Crime, Thriller, Sci-Fi, Horror, Action, Romance, Mystery)
- Search any show by name
- Show detail modal with synopsis, ratings, network, runtime, and status
- Watchlist with localStorage persistence
- Airing/Ended status badges

## API

- [TVMaze API](https://www.tvmaze.com/api) — free, no API key required

## Run

```
open index.html
```

No build step. Open the file and it works.
